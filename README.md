# BlackHole
Awesome Music Player made with Flutter

<h3>Features</h3>
Music Search<br>
Top Playlists<br>
Local and Global Top 200 songs<br>
Streaming (320kbps)<br>
Add Songs to Favorite<br>
Download Support (320kbps with ID3 tags)<br>
Play Online as well as Offline Songs<br>
